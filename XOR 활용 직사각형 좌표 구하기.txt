// XOR 비트 연산을 이용하면 동일 값을 제거할 수 있다

XOR 비트 연산자를 활용하면, 
여러 개의 값 중에서 동일한 값들을 제거할 수 있다
예를 들어,
3개의 값이 존재하고 그 중에서 2개의 값이 동일하고(동일한 값이 쌍을 이루어야 함)
나머지 하나의 값이 다른 경우,
XOR 비트 연산 2번을 통해 나머지 하나의 다른 값을 추출할 수 있다

카카오 블라인드 모의테스트 중
직사각형의 나머지 좌표 구하기에서 써먹음

class Solution {
    public int[] solution(int[][] v) {
        int[] answer = new int[2];
        
        answer[0] = v[0][0]^v[1][0]^v[2][0];
        answer[1] = v[0][1]^v[1][1]^v[2][1];

        return answer;
    }
}
